# CSC 193A-01: Assignment 5 - Fun with JavaScript

**Student Name:** Anh Ha

## Assignment Overview
This assignment demonstrates basic JavaScript DOM manipulation and event handling. Users can enter text into a textarea and use various UI controls to modify the text's appearance and content.

### Features
* **Bigger!:** Increases the font size of the text to 24pt.
* **FancyShmancy:** Applies bold, blue, and underlined styling to the text.
* **BoringBetty:** Removes all special styling, returning text to normal.
* **Moo:** Uppercases all text and appends "-MOO" to the end of each sentence.
